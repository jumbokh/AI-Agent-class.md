# **ACR122U NFC 技術與物聯網應用實務課程**  
**教學大綱與考核標準**

## **課程基本信息**
- **課程名稱**：ACR122U NFC 技術與物聯網應用實務
- **適用年級**：大學二、三年級（物聯網系）
- **學分數**：1 學分（或 2 學分專題實作）
- **先修課程**：Python 程式設計、計算機網路基礎
- **課程類型**：專業選修 / 實作課程

---

## **課程目標**

### **知識目標**
1. 理解 NFC/RFID 技術原理與通訊協議（ISO 14443）
2. 掌握 PC/SC 架構與 APDU 指令集
3. 熟悉 MIFARE Classic 與 NTAG 的記憶體結構與安全機制
4. 了解 NDEF 數據格式與物聯網應用場景

### **技能目標**
1. 能使用 Python 操作 ACR122U 進行卡片讀寫
2. 能設計並實現基於 NFC 的物聯網應用原型
3. 能整合 Web 框架建立完整的監控系統
4. 具備硬體故障排查與系統除錯能力

### **素養目標**
1. 培養物聯網系統整合思維
2. 建立資訊安全與倫理意識
3. 發展跨領域應用創新能力

---

## **課程進度安排（共 6 次實驗）**

| 週次 | 實驗主題 | 核心內容 | 生物聯網應用連結 |
|------|----------|----------|------------------|
| 1 | 實驗環境建立與基礎通訊 | PC/SC 架構、ATR/UID 讀取、APDU 基礎 | 設備身份識別與註冊 |
| 2 | MIFARE Classic 記憶體操作 | 扇區認證、區塊讀寫、安全機制 | 實驗設備權限管理 |
| 3 | 數據格式化與交易系統 | BAL= 格式設計、讀-改-寫模式 | 小額支付與資源計費 |
| 4 | NTAG NDEF 與智能標籤 | TLV 結構、NDEF 解析、Text/URI 記錄 | 樣本追踪與資產管理 |
| 5 | 安全機制與金鑰管理 | Sector Trailer 操作、存取控制位元 | 多層級權限系統 |
| 6 | Web 系統整合與應用開發 | Flask 框架、多執行緒、即時監控 | 物聯網數據可視化 |

## **期末專題建議方向**

### **生物聯網應用專題**
1. **智能實驗室管理系統**
   - 門禁控制 + 設備使用計費 + 人員權限管理
   - 使用 MIFARE Classic 實現多層級權限

2. **生物樣本追踪系統**
   - NTAG 標籤標記樣本管 + NDEF 記錄樣本信息
   - Web 系統記錄樣本流轉歷程

3. **設備維護與資產管理**
   - NFC 標籤記錄設備維護歷史
   - 掃描即顯示設備狀態與維護計劃

4. **溫室環境數據采集系統**
   - 工作人員 NFC 卡記錄數據采集任務
   - 與環境感測器數據整合可視化

## **安全與倫理規範**

### **必須遵守的準則**
1. **僅操作自有卡片**，不得測試任何在運系統
2. **備份重要數據**，避免操作失誤導致損失
3. **尊重隱私權**，不得未經授權讀取他人卡片
4. **學術誠信**，獨立完成實驗與專題開發

### **違規後果**
- 違反安全規範：該次實驗成績為 0 分
- 情節嚴重者：學期成績不及格並依校規處理
